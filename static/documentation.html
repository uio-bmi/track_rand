Overview of randomization algorithms

Shuffle genomic locations within each track (no preservation of distances between genomic regions)

---
# Track1.bed

chr1    0   100 a1  1   +
chr1    100 200 a2  2   +

# Track2.bed

chr1    200 300 a1  1   +
chr1    300 400 a2  2   +

# Track3.bed

chr1    400 500 a1  1   +
chr1    500 600 a2  2   +

---
# Track1_shuffled.bed

chr1    998 1098    a1  1   +
chr1    2709    2809    a2  2   +

# Track2_shuffled.bed

chr1    22103   22203   a1  1   +
chr1    10101   10201   a2  2   +

# Track3_shuffled.bed

chr1    290692  290792  a1  1   +
